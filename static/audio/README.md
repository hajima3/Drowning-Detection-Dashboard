# Audio Alerts

Place your audio alert files here:

- `level1.mp3` - Plays when Level 1 warning is triggered (unsafe movement)
- `level2.mp3` - Plays when Level 2 emergency is triggered (drowning detected)

## Supported Formats
- MP3 (recommended)
- WAV
- OGG

## Audio Guidelines
- Keep files under 5 seconds for quick alerts
- Use clear, distinct sounds for each level
- Test volume levels before deployment
